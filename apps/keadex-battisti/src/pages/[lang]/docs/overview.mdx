This section provides detailed documentation (_operational_, _functional_, _architectural_ and _technical_) about the projects implemented in Keadex.

Keadex documentation is broken down into functional subsections, one for each project:

## Keadex Mina

Application, built with [Tauri](https://tauri.app/) and Wasm, to easily code and organize at a scale [C4 Model](https://c4model.com/) diagrams.

## C4 Model UI Kit

Typescript library that provides [Fabric.js](https://github.com/fabricjs/fabric.js) UI components and utilities to render [C4 Model](https://c4model.com/) diagrams.
