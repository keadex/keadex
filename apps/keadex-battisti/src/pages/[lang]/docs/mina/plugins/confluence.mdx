# Mina Confluence Plugin

The Mina Confluence Plugins enable the inclusion of interactive C4 Model diagrams created with Keadex Mina in Confluence pages.

Follow these steps to start using the Mina Confluence Plugin:

1. Find and install the **Keadex Mina** macro from the Marketplace in your Confluence application.
    - Atlassian Marketplace: [Link](https://marketplace.atlassian.com/)
    - Atlassian Documentation: [Link](https://confluence.atlassian.com/upm/finding-new-marketplace-apps-302812383.html)
2. Add the **Keadex Mina** macro in a Confluence page:

![Step1](/img/mina/add-confluence-step1.png)

3. Open the macro configuration view:

![Step2](/img/mina/add-confluence-step2.png)

4. Fill the mandatory diagram configurations. E.g.:
    - Mina project root URL: https://raw.githubusercontent.com/keadex/keadex/main/examples/mina-react-example/demo_mina_project/Mina%20Demo
    - Mina diagram URL: https://raw.githubusercontent.com/keadex/keadex/main/examples/mina-react-example/demo_mina_project/Mina%20Demo/diagrams/system-context/demo-diagram

![Step3](/img/mina/add-confluence-step3.png)
