# Mina Confluence Plugin

The Mina Confluence Plugins enable the inclusion of interactive C4 Model diagrams created with Keadex Mina in Confluence pages.

## Usage

### 1. Install the plugin

Find and install the **Keadex Mina** macro from the Marketplace in your Confluence application.

- Atlassian Marketplace: [Link](https://marketplace.atlassian.com/1233762/)

### 2. Import the plugin

Add the **Keadex Mina** macro in a Confluence page:

![Step1](/img/mina/add-confluence-step1.png)

### 3. Configure the plugin

Open the macro configuration view:

![Step2](/img/mina/add-confluence-step2.png)

### 4. Add the diagram data

Fill the mandatory diagram configurations. E.g.:

- Mina project root URL: https://raw.githubusercontent.com/keadex/keadex/main/examples/mina-react-example/demo_mina_project/mina-demo
- Mina diagram URL: https://raw.githubusercontent.com/keadex/keadex/main/examples/mina-react-example/demo_mina_project/mina-demo/diagrams/system-context/demo-diagram
