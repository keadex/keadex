## Quick Overview

Mina provides an **MCP Server** that enables AI applications to connect to Keadex Mina architectural diagrams.

By using the Keadex Mina MCP Server, AI applications such as Claude or ChatGPT can access Mina diagrams, retrieve key information, and perform related tasks.

For more details, please refer to the [Mina MCP Server documentation](http://keadex.dev/en/docs/mina/features/mcp-server).
